# MeMail Privacy Policy

Last updated: March 31, 2026

## Overview

MeMail is a Chrome extension that helps you draft Gmail replies in your own voice using OpenAI.

## Data Collection

MeMail does not run any developer-controlled servers and does not collect or store your personal data.

## Email Content

When you click MeMail, the selected email content is sent transiently to OpenAI to generate a reply using your own API key.  
MeMail does not store your email content.

## API Key Storage

Your OpenAI API key is stored locally in your browser using `chrome.storage.local`.  
It does not leave your device except when used to make requests directly to OpenAI.

## Third Parties

MeMail sends data only to OpenAI for reply generation.  
OpenAI privacy policy: [https://openai.com/privacy](https://openai.com/privacy)
