# MeMail Privacy Policy
Last updated: April 2026

MeMail is a Chrome extension that generates email reply drafts inside Gmail using AI.

## Data We Collect
None. MeMail does not collect, store, or transmit any user data to any developer-controlled server.

## Email Content
The content of emails you reply to is sent directly to your chosen AI provider (OpenAI, Anthropic, Gemini, or DeepSeek) using your own API key. This is processed transiently and never stored by MeMail.

## API Keys
Your API keys are stored locally in your browser using Chrome's built-in storage. They are never sent to any server other than your chosen AI provider.

## Gmail Access
MeMail requests read-only access to your sent emails solely to learn your writing style. This data is processed locally and never sent to any MeMail server.
